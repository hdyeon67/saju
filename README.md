# 사주 풀이

생년월일을 입력하면 **만세력 라이브러리**로 사주팔자(년·월·일·시주)를 계산하고,
**Claude AI**가 자연어로 풀이해 주는 로컬 개인용 웹사이트입니다.

- AI 비용은 API 종량 과금이 아니라 **내 Claude 구독(Pro/Max)** 으로 사용합니다 (OAuth 토큰 방식).
- 회원가입·결제 없이 내 컴퓨터에서만 실행합니다.

## 동작 방식 (쉽게)

1. 화면에서 생년월일·시각·성별·양/음력을 입력합니다.
2. 뒤에서 도는 서버(Next.js API Route)가
   - `@fullstackfamily/manseryeok` 라이브러리로 사주팔자를 **정확히 계산**하고
   - 그 결과를 Claude에게 넘겨 **풀이 글을 생성**합니다.
3. 풀이 글이 실시간(스트리밍)으로 화면에 나타납니다.

> AI는 브라우저에서 직접 부를 수 없어, 서버가 대신 호출합니다. 계산 자체는 AI가 아니라 규칙 기반 라이브러리가 합니다.

## 준비 (최초 1회)

1. **Node.js 18+** 설치 확인 (`node -v`)
2. **Claude Code CLI 설치**
   ```bash
   npm i -g @anthropic-ai/claude-code
   ```
3. **구독 토큰 발급** (Pro/Max 구독 필요)
   ```bash
   claude setup-token
   ```
   출력된 `sk-ant-oat01-...` 값을 복사합니다. (이 토큰은 어디에도 저장되지 않으니 직접 복사하세요.)
4. **토큰 등록**: 프로젝트 루트의 `.env.local` 파일을 열어 붙여넣습니다.
   ```
   CLAUDE_CODE_OAUTH_TOKEN=sk-ant-oat01-여기에_붙여넣기
   ```

## 실행

```bash
npm install       # 최초 1회 (이미 설치돼 있으면 생략)
npm run dev
```

브라우저에서 http://localhost:3000 접속 → 생년월일 입력 → **사주 풀이 보기**.

## 참고 / 주의

- 만세력 지원 범위는 **1900~2050년** 입니다.
- 태어난 시각을 모르면 **"시간 모름"** 을 체크하세요. 시주를 제외하고 나머지로 풀이합니다.
- 음력 생일이면 **달력을 "음력"** 으로 바꾸고, 윤달이면 체크하세요. (내부에서 양력으로 변환 후 계산)
- 풀이 결과는 **참고용**이며 재미로 즐겨 주세요.
- OAuth 토큰은 원칙적으로 Claude Code·claude.ai 용도입니다. **개인 로컬 사용**은 공식 지원 경로지만, 외부 서비스로 배포·판매 시 약관을 확인하세요.
