## 1. 개요 (무엇을 만들었나)

생년월일을 입력하면 **만세력으로 사주팔자를 계산**하고, **Claude AI가 풀이**해 주는 웹 서비스.
현재는 회원가입·결제 없이 **로컬(내 PC)에서 동작**하며, AI 비용은 **내 Claude 구독(OAuth 토큰)** 으로 사용.

동작 흐름:
1. 사용자가 생년월일·시각·성별·양/음력 입력
2. 서버(Next.js API Route)가 `@fullstackfamily/manseryeok`로 사주팔자 계산
3. 계산 결과 + 오늘 날짜를 Claude(Sonnet)에게 전달 → 풀이 생성(스트리밍)
4. 화면에 사주팔자표 · 오늘의 행운 · 풀이를 표시

---

## 2. 완료된 기능

### 입력
- 양력/음력 선택(음력은 내부에서 양력 변환, 윤달 지원), 성별
- 년·월·일 **드롭다운**(선택한 월에 맞춰 일수 자동 조정)
- 태어난 시각(시/분), "시간 모름" 지원(시주 제외)

### 사주팔자 표시
- 천간/지지를 세로로 쌓은 전통식 4기둥(시·일·월·년) + 한글/한자
- 각 기둥 의미 설명(자녀·말년 / 나·배우자 / 부모·사회 / 조상·초년)
- 사주 개념 설명 문구

### 오늘의 행운 (사주팔자와 풀이 사이, AI 없이 즉시 계산)
- 나의 띠(십이간지) + 3문장 해석
- 별자리(양력 생일 기준) + 해석
- 행운의 색 / 방향 / 음식 / 아이템 / 시간 / 어울리는 날씨
- 행운의 숫자 6개, 오늘의 기운 지수(점수+등급)
- 생년월일 + 오늘 날짜 기반 → 매일 바뀌되 같은 날엔 동일

### AI 풀이 (11개 섹션, 스트리밍)
- 성향: 총평 / 성격과 기질 / 재물운 / 연애와 궁합 / 직업과 적성 / 건강
- 시기별: 오늘 / 이번 주 / 이번 달 / 올해 / 내년 (오늘 날짜 기준, 달력 아이콘 강조)
- 항목별 "복사" 버튼 + "전체 복사"
- 생성 중 로딩 애니메이션(회전 링 + 순환 문구 + 스켈레톤)

### 저장 · 공유 (맨 아래)
- 결과 전체를 **PNG 이미지로 저장**(html-to-image)
- 기본 공유 시트(이미지 첨부) / 링크 복사 / X(트위터) 공유

### 디자인
- 동양풍 단청·오방색 테마(짙은 남색 + 금빛 + 오방색 포인트)
- 한글 명조(제목)·고운돋움(본문) 폰트
- 로고 인장 은은한 발광 애니메이션
- 다크 테마에 맞춘 드롭다운/체크박스(커스텀)

## 3. 기술 스택 · 구조

- **프레임워크**: Next.js 16 (App Router) + React 19 + TypeScript
- **스타일**: Tailwind CSS v4 + shadcn 계열 컴포넌트
- **사주 계산**: `@fullstackfamily/manseryeok` (MIT, 1900~2050)
- **AI**: `@anthropic-ai/claude-agent-sdk` (모델: Sonnet)
- **이미지 저장**: `html-to-image`

```
app/
  page.tsx            # 입력 폼 + 결과(사주팔자/오늘의행운/풀이) + 저장·공유
  api/saju/route.ts   # 서버: 사주 계산 → 헤더 전달 + 풀이 스트리밍
  layout.tsx, globals.css
lib/
  saju.ts             # 만세력 래핑(입력→사주팔자)
  reading.ts          # 프롬프트 구성 + Agent SDK 호출
components/ui/         # button, card, input, select, label, checkbox 등
```
